HTML Elements

  • <title>

    The <title> HTML element defines the document's title that is shown in a browser's title bar or a page's tab. It only contains text; tags within the element are ignored.

    This is neither an inline nor block level element

  • <p>

    The <p> HTML element represents a paragraph. Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation, but HTML paragraphs can be any structural grouping of related content, such as images or form fields.

    Paragraphs are block-level elements, and notably will automatically close if another block-level element is parsed before the closing </p> tag.

    This is a block level element

  • <code>

    The <code> HTML element displays its contents styled in a fashion intended to indicate that the text is a short fragment of computer code. By default, the content text is displayed using the user agent's default monospace font.

    This is an inline element

  • <img>

    The <img> HTML element embeds an image into the document.

    This is an inline element

HTML Character Entities

  • & l t ;

    Use this HTML character entity to type out the < symbol in your website when you would like to show a HTML element.

    When i use this symbol it looks like this <

  • & g t ;

    Use this HTML character entity to type out the > symbol in your website when you would like to show a HTML element.

    When i use this symbol it looks like this >

  • cent symbol - & c e n t ;

    Use this HTML character entity to put the cent symbol on your website.

    When i use this symbol it looks like this ¢

CSS Selectors

  • type

    The CSS type selector matches elements by node name. In other words, it selects all elements of the given type within a document.

  • .class

    The .class selector selects elements with a specific class attribute.

  • #id

    The #id selector styles the element with the specified id.

  • :nth-child

    The :nth-child(n) selector matches every element that is the nth child of its parent.

CSS Properties

  • background-color

    The background-color property sets the background color of an element.

    A couple possible values are tomato, gold, lightblue and pink!

  • font-family

    The font-family property specifies the font for an element.

    A couple possible values are times or arial

  • font-size

    The font-size property sets the size of the text.

    A couple possible values are 100px or 14px
